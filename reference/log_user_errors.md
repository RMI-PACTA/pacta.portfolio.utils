# Log user errors

Log user errors

## Usage

``` r
log_user_errors(
  error_name,
  suggested_action = "",
  description = "",
  error_file = file.path(.GlobalEnv$outputs_path, .GlobalEnv$portfolio_name_ref_all,
    "report", "user_errors.json"),
  immediate = FALSE
)
```

## Arguments

- error_name:

  A parameter

- suggested_action:

  A parameter

- description:

  A parameter

- error_file:

  A parameter

- immediate:

  A parameter

## Value

Return value
